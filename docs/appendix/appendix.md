## 附录1:返回值Code说明

| code   | 说明               |
|--------|------------------|
| 0      | 正常返回,成功      |
| 400xxx | 参数错误           |
| 400000 | 参数不符合规范     |
| 401xxx | 需要身份验证       |
| 403xxx | 访问被拒绝         |
| 404xxx | 未找到资源         |
| 500xxx | 服务器出错         |
| 500000 | 服务器内部错误     |
| 503xxx | 服务器维护或者宕机 |
